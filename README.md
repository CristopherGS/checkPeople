# Explorador de Registros

Web estática para abrir el JSON, buscar registros, filtrar por campos clave, ordenar columnas y ver la foto de cada persona.

## Uso

1. Abre `index.html` en el navegador o sírvelo con un servidor local.
2. Si la carga automática no funciona, usa el botón **Cargar JSON** y selecciona `________________________________________.json`.
3. Haz clic en los encabezados para ordenar y en **Columnas visibles** para mostrar u ocultar campos.

## Qué incluye

- Buscador global.
- Filtros por departamento, municipio, estado laboral, estado civil y rango de edad.
- Panel lateral con foto y detalle completo del registro seleccionado.
- Columnas configurables y ordenamiento por encabezado.
